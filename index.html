🔍 ¿Cómo funciona? ▼
🎟️
Esto es solo una guía, ignorala sin compromiso
Son regalos sugeridos, solo para organizar y evitar repeticiones dentro de lo posible, pero aprovechamos toda ayuda y como es algo nuevo para todos aprovechamos a dejar lo que nos viene bien.
🎟️
Código de un solo uso
Cada invitado recibe un link con un código único. Una vez que reservás un regalo, el código queda quemado y no podés volver a usarlo.
🕶️
Reserva 100% anónima
El sistema no registra qué código eligió qué regalo. Solo el sistema sabe si el código fue usado o no, yo (fer) lo hice para que sea anónimo. Nadie sabe quién reservó qué. (así que no jueguen que me lo rompen, je)
🚫
Sin duplicados
Cuando reservás un regalo queda marcado para todas al instante. No puede reservarlo otra persona al mismo tiempo.
☁️
Sin base de datos propia
Todo se guarda en Cloudflare Workers KV. No hay servidor propio ni base de datos con tus datos personales.
